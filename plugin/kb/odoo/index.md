# Odoo Knowledge Base

> Odoo 14-17 reference for setup, module development, security, integrations, and migration.

## Overview

This domain is organized into:
- `concepts/` for architecture and core principles
- `patterns/` for implementation-ready recipes
- `quick-reference.md` for fast lookup

- Modules system and lifecycle
- Models and ORM patterns
- Views and UI composition
- Security (ACL + record rules)
- Workflow and automation
- API and integrations

### Implementation Patterns
- Custom module scaffolding
- Models + views working together
- CSV migration/import
- QWeb reports
